Preguntas frecuentes

Resolvemos tus dudas sobre el servicio

¿Cómo funciona el sistema de reserva de citas?

Tu sitio web incluirá un calendario donde los clientes podrán ver tu disponibilidad y reservar turnos directamente. El sistema se puede sincronizar con tu Google Calendar o Outlook para mantener todo actualizado automáticamente. Además, recibirás notificaciones por correo cada vez que se confirme una nueva reserva.

¿Puedo ofrecer consultas virtuales?

SSí. Podés integrar plataformas como Zoom, Google Meet o Microsoft Teams. Cuando un cliente agenda una cita virtual, el sistema genera automáticamente un enlace de videollamada y lo incluye en el correo de confirmación.

¿Qué tipo de profesionales pueden usar este servicio?

Está pensado para psicólogos, abogados, nutricionistas, médicos, coaches, contadores, diseñadores, consultores y otros profesionales independientes que necesiten una presencia online profesional con sistema de citas.

¿Puedo actualizar mi perfil y servicios?

Sí. Vas a contar con un panel de administración simple donde podés editar tu perfil, agregar especialidades, cambiar precios, publicar en tu blog y gestionar tu disponibilidad sin necesidad de conocimientos técnicos.

¿Qué pasa después del primer año?

El primer año incluye dominio y hosting. Una vez cumplido ese período, necesitás renovarlos para que tu sitio siga funcionando. Te avisaremos con anticipación y podemos ayudarte a gestionar la renovación si lo deseás.

¿Es seguro el área privada para documentos?

Sí. Cada cliente accede a su área privada con credenciales únicas, y todos los datos se transmiten mediante conexiones cifradas (HTTPS). Además, podés decidir qué archivos están disponibles para cada cliente, garantizando privacidad y control total.
